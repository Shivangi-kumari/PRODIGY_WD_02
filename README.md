⏱️ Simple Stopwatch

This is a basic stopwatch web app built using HTML, CSS, and JavaScript. It includes Start, Pause, Reset, and Lap features.

 📋 Features

- ✅ Start the stopwatch
- ⏸️ Pause and resume
- 🏁 Record lap times
- 🔄 Reset to zero
